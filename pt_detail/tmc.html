PROJECT DETAILS

목표

Navigation

토요타 및 렉서스 차량용 네비게이션 Agent 서버를 구축하여 사용자에게 신속하고 정확한 도로 정보 제공

기간 : 2023.07 ~ 2024.04
역할 : Agent 서버 관리자 및 개발자
기술 스택 : SpringBoot 3, Java 17, Mysql 8

과정

Toyota,Lexus Navigation Service

Spring Boot 3.15 기반
Toyota,Lexus 차량용 네비게이션
Agent 서버 구축

1. 서비스 기능 구현

  • 서비스 버전 업그레이드
    • 레거시 시스템(Spring Boot 1.5)을 최신 버전(Spring Boot 3.15)으로 업데이트
    • 라이브러리 충돌과 호환성 문제 해결
  • 차량용 네비게이션 기능 커스터마이징
    • 토요타와 렉서스 차량용 네비게이션에 추가 요구된 기능 개발
    • 기존 API를 수정하여 토요타와 렉서스 차량용 네비게이션에 맞게 적용

2. 이슈 관리

  • QA 팀에서 Jira,Slack등으로 문의가 들어오는 이슈를 신속하게 로그 기반으로 식별하고 문제 해결
  • 엔진 및 단말기 관련 이슈를 담당자들과 협력하여 문제 해결

3. 자동 배포 시스템 운영

  • Gradle을 이용한 빌드 및 배포 자동화로 배포 시간 70% 이상 단축,
    • 서비스 JAR 파일 백업 후 안정적으로 서버에 배포
    • systemctl을 사용한 서비스 자동 재기동으로 시스템 가동률 99.9% 유지

4. 문제 해결

  • 엔진 통신 문제 해결
    • 문제 발생 시 서버에서 로그를 기반으로 선 검토
    • 요청 시간, 통신 방식, 헤더, 바디, 파라미터 정보를 정리하여 엔진 담당자에게 문의
    • 엔진 담당자와 협력하여 해결
  • Multipart 통신 에러
    • 파일 데이터 전송 중 Multipart 방식과 Chunk 방식 간의 불일치 문제 발생
    • 데이터 전송 방식을 Chunk 방식으로 조정, Chunk 크기를 명시적으로 설정하여 문제 해결

성과

팀원들과 협력하여 각자의 전문성을 활용, 이슈와 문제를 효과적으로 해결하고 서비스 품질을 향상시켰습니다.

2. 이슈 해결률 100% 달성

Mantis, Teams 활용 이슈 식별 및 해결 프로세스 개선

3.신속한 서버 문제 해결

로그 기반 서버/클라이언트 문제 진단 및 해결
